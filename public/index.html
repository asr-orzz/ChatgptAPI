Session Guide

This flow avoids depending on noVNC to refresh a login on the server.

  1. Run this repo locally and execute npm run login to log into ChatGPT in a normal browser.
  2. Take the generated sessions/auth.json file from that local run.
  3. Upload or paste it here and let the server verify the session against ChatGPT.
  4. Use /ask or export the saved session again when you need a backup.

The legacy remote login endpoints still exist, but this page no longer relies on the noVNC preview as the main path.
